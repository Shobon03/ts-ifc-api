## Interface do Sistema

### Menu Principal

O sistema possui as seguintes páginas principais:

```typescript
const navigation = {
  "Home": "Página inicial com visão geral",
  "Sobre": "Informações sobre o projeto",
  "Geração de IFC": "Conversão simples (Revit/Archicad → IFC)",
  "Transformação de Modelos": "Conversões em cadeia (ex: Revit → IFC → Archicad)"
}
```

### Elementos Comuns da Interface

Todas as páginas de conversão compartilham elementos similares:

1. **Indicadores de Status**
   - 🟢 **WebSocket**: Conexão em tempo real
   - 🟢/🔴 **Status dos Plugins**: Revit e Archicad
   - 🌙/☀️ **Modo Escuro/Claro**: Alternância de tema

2. **Área de Upload**
   - Drag-and-drop (arrastar e soltar)
   - Botão "Escolher arquivo"
   - Indicação do tipo de arquivo aceito
   - Exibição do arquivo selecionado (nome e tamanho)

3. **Botão de Conversão**
   - Ativa após selecionar arquivo válido
   - Inicia o processamento

4. **Barra de Progresso**
   - Aparece durante a conversão
   - Mostra porcentagem (0-100%)
   - Mensagens de status em tempo real
   - Plugin em uso (Revit ou Archicad)

5. **Indicador de Status WebSocket**
   - 🟢 **Verde**: Conectado (tudo OK)
   - 🟡 **Amarelo**: Reconectando (tentativa N/5)
   - 🔴 **Vermelho**: Desconectado (verifique conexão)

## Sua Primeira Conversão

### Opção 1: Conversão Simples (Revit → IFC)

Vamos fazer a conversão mais simples: transformar um arquivo Revit em IFC.

<Steps>

<Step>

### Navegue para Geração de IFC

1. No menu superior, clique em **"Geração de IFC"**
2. Você verá a página de upload

<Callout type="info">
  Verifique o **status dos plugins** no topo da página. Para conversão Revit, você precisa que o indicador mostre "🟢 Revit: Conectado".
</Callout>

</Step>

<Step>

### Selecione o Arquivo

**Método 1: Arrastar e Soltar**
- Arraste o arquivo `.rvt` para a área indicada
- Solte quando a área ficar destacada

**Método 2: Clicar para Selecionar**
- Clique no botão "Escolher arquivo" ou na área de upload
- Navegue até o arquivo `.rvt`
- Selecione e clique "Abrir"

**Requisitos do arquivo:**
- Formato: `.rvt` (Revit) ou `.pln` (Archicad)
- Tamanho máximo: 100MB
- Arquivo deve abrir normalmente no software original

Após selecionar, você verá:
```
📄 meu-projeto.rvt (25.6 MB)
```

</Step>

<Step>

### Inicie a Conversão

1. Clique no botão **"Converter para IFC"**
2. A conversão inicia imediatamente
3. Você NÃO pode fechar a página durante o processo

<Callout type="warn">
  Mantenha a aba do navegador aberta. Se fechar, você perde o acompanhamento do progresso (mas a conversão continua no servidor).
</Callout>

</Step>

<Step>

### Acompanhe o Progresso

A barra de progresso mostrará as etapas em tempo real:

**Para arquivos Revit (.rvt):**
```typescript
0-20%  → "Enviando arquivo para o servidor..."
20-40% → "Enviando para Autodesk Forge..."
40-80% → "Convertendo para IFC (pode demorar alguns minutos)..."
80-95% → "Baixando resultado..."
95-100% → "Finalizando..."
```

**Para arquivos Archicad (.pln):**
```typescript
0-30%  → "Enviando arquivo para o servidor..."
30-50% → "Processando no servidor Python..."
50-70% → "Abrindo no Archicad..."
70-90% → "Exportando IFC..."
90-100% → "Preparando download..."
```

**Tempo estimado:**
- Pequeno (< 10MB): 1-2 minutos
- Médio (10-50MB): 3-5 minutos
- Grande (50-100MB): 5-10 minutos

</Step>

<Step>

### Baixe o Arquivo

Quando a conversão atingir **100%**:

1. O arquivo `.ifc` será **baixado automaticamente** (verifique a pasta Downloads)
2. Você verá uma mensagem de sucesso:
   ```
   ✅ Concluído!
   📄 meu-projeto.ifc (18.2 MB)
   ```
3. Um botão **"Fazer download"** aparecerá (caso precise baixar novamente)

</Step>

</Steps>

### Opção 2: Conversão em Cadeia (Revit → Archicad)

Agora vamos fazer uma conversão mais avançada: transformar um arquivo Revit diretamente em Archicad.

<Steps>

<Step>

### Navegue para Transformação de Modelos

1. No menu superior, clique em **"Transformação de Modelos"**
2. Você verá uma interface mais avançada

<Callout type="success">
  Esta página permite conversões automáticas em cadeia! Por exemplo: Revit → IFC → Archicad acontece automaticamente.
</Callout>

</Step>

<Step>

### Selecione o Arquivo de Origem

- Faça upload do arquivo `.rvt` (mesmo processo da Opção 1)
- O sistema detecta automaticamente o formato

</Step>

<Step>

### Escolha o Formato de Destino

Após o upload, você verá checkboxes:

**Se você enviou `.rvt`:**
- ☑️ **Archicad (.pln)** - Converte para Archicad

**Se você enviou `.pln`:**
- ☑️ **Revit (.rvt)** - Converte para Revit

**Se você enviou `.ifc`:**
- ☑️ **Revit (.rvt)**
- ☑️ **Archicad (.pln)**
- Pode selecionar ambos!

Marque as opções desejadas.

</Step>

<Step>

### Visualize o Fluxo

O sistema mostra uma seta indicando o fluxo:

**Exemplo 1:**
```
Revit → IFC → Archicad
```

**Exemplo 2:**
```
IFC → Revit
IFC → Archicad
```

</Step>

<Step>

### Inicie a Transformação

1. Clique em **"Converter Modelo"**
2. Você verá **múltiplas barras de progresso**:
   - Uma para cada etapa da conversão
   - Executam em sequência ou paralelo (depende do caso)

**Conversão em Sequência** (Revit → IFC → Archicad):
```
[████████████] 100% Revit → IFC (concluído)
[████░░░░░░░░]  40% IFC → Archicad (processando...)
```

**Conversão em Paralelo** (IFC → Revit e Archicad):
```
[████████░░░░]  75% IFC → Revit
[██████░░░░░░]  60% IFC → Archicad
```

</Step>

<Step>

### Baixe os Resultados

- Cada conversão completada oferece download independente
- Arquivos baixam automaticamente quando prontos
- Você pode iniciar nova conversão clicando "Nova Conversão"

</Step>

</Steps>

## Dicas para Iniciantes

### Preparação do Arquivo

Antes de fazer upload:

✅ **Faça**:
- Salve uma cópia do arquivo original (backup)
- Verifique se o arquivo abre normalmente no software nativo
- Remova elementos desnecessários para reduzir tamanho
- Use a última versão salva (sem erros)
- Feche o arquivo no software antes de fazer upload

❌ **Evite**:
- Arquivos corrompidos ou com erros
- Modelos extremamente complexos na primeira vez
- Arquivos de versões muito antigas dos softwares
- Arquivos que nunca foram testados antes

### Durante a Conversão

✅ **Faça**:
- Mantenha a aba do navegador aberta
- Aguarde pacientemente (pode demorar alguns minutos)
- Observe as mensagens de status
- Anote qualquer erro que aparecer
- Verifique se o indicador WebSocket está verde (🟢)

❌ **Evite**:
- Clicar múltiplas vezes no botão de conversão
- Fechar o navegador durante o processo
- Fazer upload de outro arquivo enquanto um está processando
- Atualizar a página (F5) durante a conversão
- Desligar o computador

### Após a Conversão

✅ **Faça**:
- Verifique a pasta de Downloads do navegador
- Teste o arquivo convertido no software de destino
- Guarde uma cópia de backup
- Documente eventuais problemas encontrados
- Relate erros para a equipe

❌ **Evite**:
- Deletar o arquivo original imediatamente
- Compartilhar o arquivo convertido sem testar primeiro
- Assumir que tudo foi convertido perfeitamente sem verificar
- Confiar apenas no arquivo convertido (mantenha o original)

## Entendendo a Interface

### Indicadores de Status

#### WebSocket (Canto Superior)

| Indicador | Significado | Ação |
|-----------|-------------|------|
| 🟢 Conectado | Conexão ativa, tudo funcionando | Continue normalmente |
| 🟡 Reconectando (N/5) | Conexão perdida, tentando reconectar | Aguarde alguns segundos |
| 🔴 Desconectado | Sem conexão com servidor | Verifique internet/backend |

#### Status dos Plugins

| Indicador | Significado | O que fazer |
|-----------|-------------|-------------|
| 🟢 Revit: Conectado | Plugin Revit está ativo | Pode fazer conversões Revit |
| 🔴 Revit: Desconectado | Plugin não disponível | Conversões Revit não funcionarão |
| 🟢 Archicad: Conectado | Plugin Archicad está ativo | Pode fazer conversões Archicad |
| 🔴 Archicad: Desconectado | Plugin não disponível | Conversões Archicad não funcionarão |

<Callout type="info">
  Os plugins rodam no servidor, não no seu computador. Se um plugin estiver desconectado, contate o administrador do sistema.
</Callout>

### Mensagens de Status Comuns

| Mensagem | Significado | Normal? |
|----------|-------------|---------|
| "Enviando arquivo..." | Upload em andamento | ✅ Sim |
| "Convertendo para IFC..." | Processamento ativo | ✅ Sim (pode demorar) |
| "Conversão concluída!" | Sucesso! | ✅ Sim |
| "Erro: Formato de arquivo inválido" | Arquivo errado | ❌ Use .rvt, .pln ou .ifc |
| "Erro: Arquivo muito grande" | Excede 100MB | ❌ Reduza o tamanho |
| "Erro: WebSocket desconectado" | Sem conexão | ❌ Verifique internet |
| "Plugin não conectado" | Serviço indisponível | ❌ Contate administrador |

## Resolução Rápida de Problemas

### Upload Falha

**Sintomas:** Arquivo não carrega, mensagem de erro ao selecionar

**Possíveis causas:**
1. Arquivo muito grande (> 100MB)
2. Formato incorreto (não é .rvt, .pln ou .ifc)
3. Conexão lenta ou instável
4. Arquivo corrompido

**Soluções:**
- Verifique o tamanho: clique com botão direito → Propriedades
- Confirme a extensão do arquivo
- Tente novamente com internet melhor
- Abra o arquivo no software nativo para verificar se está OK

### Conversão Trava em X%

**Sintomas:** Progresso para em uma porcentagem e não avança

**Possíveis causas:**
1. Arquivo muito complexo (processamento demorado)
2. Servidor ocupado
3. Plugin travou
4. Problema no arquivo

**Soluções:**
- Aguarde até 10 minutos (arquivos grandes demoram)
- Recarregue a página (F5) e tente novamente
- Verifique se o WebSocket está conectado (🟢)
- Simplifique o modelo se possível
- Entre em contato com suporte se persistir

### Download Não Inicia

**Sintomas:** 100% completo mas nenhum arquivo baixa

**Possíveis causas:**
1. Bloqueio do navegador (pop-ups/downloads)
2. Pasta de downloads cheia
3. Erro na conversão (apesar dos 100%)
4. Permissões de arquivo

**Soluções:**
- Verifique configurações de download do navegador
- Libere espaço em disco
- Veja se há mensagem de erro abaixo da barra
- Tente em outro navegador
- Clique no botão "Fazer download" manualmente

### WebSocket Desconectado (🔴)

**Sintomas:** Indicador vermelho, progresso não atualiza

**Soluções:**
1. Recarregue a página (F5)
2. Verifique sua conexão com internet
3. Verifique se o backend está rodando: abra `http://localhost:3000/health`
4. Aguarde reconexão automática (até 5 tentativas)
